waybillNumber - номер накладной, строка 50 символов
name - ФИО контакта, строка 20 символов
company - Название Юр лица, строка 40 символов
phone - телефон, строка 50 символов
country - название страны, строка 50 символов
state - название области, строка 50 символов
сity - название города, строка 50 символов
address - остальная информация адреса, строка 100 символов
zip - индекс, строка 9 символов
weight - вес в кг, дробное число
volumeWeight - объемный вес, дробное число
packs - число мест, целое число
payType - тип оплаты, "inv", "csh"
payerType - 1(отправитель), 2(получатель), 3(третье лицо)
packType - тип упаковки, "le", "pl"
waybillDate - Дата в формате YYYYMMDD
senderComment - примечание отправителя, строка 300 символов
description - описание груза, строка 500 символов

АПИ возвращает данные в формате JSend

Для работы с АПИ необходимо сгенерировать ключ(token):

 

Для этого на верхней панели нажимаем кнопку "Ключ"

В открывшемся окне нажимаем кнопку "Новый" и получаем ключ для работы с АПИ, который нужно отправлять при вызове любого запроса АПИ в виде поля заголовка "TOKEN".

 

Доступные запросы:

 

web.flippost.com/api/cities - возвращает все города доступные для заказа.

web.flippost.com/api/cities/<Город> - ищет город по неполному имени города

web.flippost.com/api/cities/<Город>/<Область> - ищет город по неполному имени города и области

web.flippost.com/api/cities/<Город>/<Область>/<Страна> - ищет город по неполному имени города, области и страны

web.flippost.com/api/types - возвращает справочник типов.

ordernumber - Номер заказа.

orgcitycode - Код города пункта отправления.

orgcity - Название города пункта отправления.

sendername - Название отправителя.

senderaddress - Адрес отправителя.

sendercontactname - Ф.И.О. контакта отправителя.

sendercontactphone - Телефон контакта отправителя.

sendercontacemail - Электронная почта контакта отправителя.

senderdescription - Дополнительная информация по отправлению.

cargotype - Тип груза (0 - Не документ, 1 - Документ).

numberofpackages - Количество упаковок в отправлении.

weight - Вес отправления.

volumeweight - Объемный вес отправления.

destcitycode - Код города пункта назначения.

destcity - Название города пункта назначения.

destinationname - Название получателя.

destinationaddress - Адрес получателя.

destinationcontactname - Ф.И.О. контакта получателя.

destinationcontactphone - Телефон контакта получателя.

destinationcontacemail - Электронная почта контакта получателя.

destinationdescription - Дополнительная информация по получению.

courierarrivaldate - Дата приезда курьера.

courierarrivaltimefirst - Начало периода приезда курьера.

courierarrivaltimelast - Конец периода приезда курьера.

status - Текущий статус заказа.

 

Все запросы используют для проверки авторизации ключ в HTTP заголовке "TOKEN"
